🗑️ AdNull 테스트 페이지

이 페이지는 AdNull 크롬 확장 프로그램을 테스트하기 위한 페이지입니다.

페이지에는 총 개의 div 요소가 있습니다.

📋 테스트 방법

  1. 브라우저 툴바에서 AdNull 확장 프로그램 아이콘을 클릭하세요
  2. "모든 div 삭제" 버튼을 눌러서 모든 div가 사라지는지 확인하세요
  3. "복원하기" 버튼으로 div들이 다시 나타나는지 확인하세요
  4. "선택해서 삭제" 모드로 개별 div를 삭제해보세요

⚠️ 주의사항

이 테스트 페이지의 모든 콘텐츠는 div 요소로 구성되어 있습니다. "모든 div 삭제"를 실행하면 페이지가 완전히 비어 보일 수 있습니다.

테스트 카드 1

빨간색 테스트 div
중첩된 div 1
더 깊이 중첩된 div

테스트 카드 2

파란색 테스트 div
중첩된 div 2

테스트 카드 3

초록색 테스트 div
보라색 테스트 div

📊 통계

현재 페이지의 div 개수를 확인하려면 AdNull 확장 프로그램을 열어보세요.

  • 🎯 메인 타깃 배너 #3 - 이 요소가 제거됩니다
